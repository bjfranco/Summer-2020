Chapter 2 - Programming Basics

  • Each statement in JS ends with a newline or with a semicolon. Colons will be automatically added if forgotten.
  • JS has 7 different data types: String, Symbol, Number, Boolean, Undefined, Null.
  • The 'typeof' operator finds out the type of a value.
  • 'const' and 'let' are used to declare variables. Const indicates that a variable will not be reassigned or changed, whereas Let indicates the variable may be reassigned at some point.
  • Blocks are considered the code between curly braces {}.
  • Variables declared outside of a block are considered Global Variables. These should be limited.
  • A variable declared within a block without 'const' or 'let' has a global scope.
  • Variable names that begin with an underscore generally denote a private property or method.
  • A method is an action that a primitive data type or object can perform. It is called using the dot operator and ends in parentheses. Example: name.toUpperCase();
  • Nan is an error that indicates the result is not a number.
  • A string can be converted to a number with the Number() method. Ex: Number('10'). If a string cannot be converted such as 'hello' then the result is NaN. Another method is to multiply the string by 1 since the result will be a number.
  • A number can be converted into a string with the String() method. Ex: String(3).
  • The parseInt() method can be used to convert a string representation of a numerical number back into a number.

Chapter 3 - Arrays, Logic, & Loops

  • An array is an ordered list of values. Ex: const myArray = [].
  • An array can hold a variety of different data types.
  • The delete operator will remove an item from an array. Ex: delete myArray[3]. The value is not deleted but rather replaces with the 'undefined' value.
  • The length of an array can be found using the length property. Ex: myArray.length
  • To remove the last item on an array use the pop() method.
  • The shift() method removes the first item in the array.
  • The concat() method can be used to merge an array with one or more arrays.
  • The join() method can be used to turn the array into a string that comprises all the items in the array, separated by commas.
  • The slice() method creates a subarray.
  • The splice() method removes items from an array then inserts new items in their place.
  • The reverse() method reverses the order of an array.
  • The sort() method sorts the order of the array.
  • We can find out if an array contains a particular value using the indexOf() method to find the first occurrence of a value in an array.
  • A set is a data structure that represents a collection of unique values, so it cannot include any duplicate values. Ex: const list = new Set();
  • The add() method adds values to a set.
  • Maps are used to store data in key-value pairs.
